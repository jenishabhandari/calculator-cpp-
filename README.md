# 🧮 Calculator in C++

A basic command-line calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

## 📋 Features
- Simple terminal interface
- Handles invalid inputs (like divide by zero)
- Continuous loop until user exits

## 🧑‍💻 How to Run

```bash
g++ calculator.cpp -o calculator
./calculator
